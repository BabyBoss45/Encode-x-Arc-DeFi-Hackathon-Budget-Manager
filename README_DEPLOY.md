# Деплой проекта

## Быстрый старт

**Просто следуйте инструкции в `RAILWAY_QUICK_START.md`**

## Что нужно знать

- ✅ **Backend и Frontend** деплоятся на Railway
- ✅ **База данных** остается на ngrok (как настроено)
- ✅ Автоматическое обновление при push в GitHub

## Файлы для деплоя

- `Procfile` - команда запуска для Railway
- `runtime.txt` - версия Python
- `RAILWAY_QUICK_START.md` - пошаговая инструкция

## Локальный запуск

Для локального запуска используйте:
```powershell
cd backend
python main.py
```

В другом терминале:
```powershell
cd src
python frontend.py
```
